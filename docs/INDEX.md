## Objectif

## Convention de contribution

1. Toute nouvelle classe/fonction publique doit être documentée dans le fichier `.md` correspondant.
2. Toute création de fichier Python dans `xcore/` doit être accompagnée d’une page de doc dédiée.
3. Mettre à jour l’`INDEX.md` du module concerné.
