Une structure découplée

Inspirée de la hiérarchie ./xcore ./manager ./plugins

01. XCORE

./xcore/appcfg.py

Socle de configuration centralisé. Gère les événements, les middlewares de sécurité et l'orchestration globale de l'application FastAPI.

  • ✓ Event-driven architecture
  • ✓ Global Access Control

02. MANAGER

./manager/plManager/

Système de découverte automatique. Loader, Validator et Reloader permettent d'injecter des fonctionnalités à chaud sans redémarrer.

  • ✓ Runtime Plugin Injection
  • ✓ Database Auto-migrations

03. MICROUI

./frontend/microui/

Votre Design System programmable en Python. Génère du HTML Tailwind/DaisyUI côté serveur pour une interactivité maximale via HTMX.

  • ✓ Server-Driven Components
  • ✓ Zéro JS boilerplate

Isolation & Autonomie

Chaque plugin (comme votre LibraryHub) possède sa propre autonomie : modèles SQLAlchemy, services métiers, et agents IA isolés.
